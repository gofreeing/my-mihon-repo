# Miraraw Mihon Extension

A Mihon extension for reading manga/manhwa from [Miraraw](https://miraraw.com).

## Installation

1. Open Mihon
2. Go to Settings → Browse → Extension repositories
3. Add this repository URL: `https://raw.githubusercontent.com/gofreeing/my-mihon-repo/main/index.min.json`
4. The Miraraw extension should appear in the extension list
5. Install and enable the extension

## Features

- Browse popular manga
- Search for manga
- View latest updates
- Read manga chapters

## Development

To build this extension, you need:

- Android SDK
- Gradle
- Java 8 or higher

### Building

```bash
./gradlew assembleDebug
```

The APK will be generated in `build/outputs/apk/debug/`
